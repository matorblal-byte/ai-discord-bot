# ai-discord-bot
ai bot on discord (like shapes inc) <br>
<br>
Download python <br>
Open cmd, run these (Windows only) <br>
py -m pip install discord <br>
py -m pip install torch <br>
py -m pip install llama-cpp-python --extra-index-url abetlen.github.io <br>
<br>
If downloading llama doesn't work, download these: <br>
Download VS 2022 here: https://aka.ms/vs/17/release/vs_community.exe <br>
Download CUDA 12.1 here: https://developer.nvidia.com/cuda-12-1-0-download-archive?target_os=Windows&target_arch=x86_64&target_version=11&target_type=exe_network <br>
Then try again <br>
<br>
Download the AI: https://huggingface.co/TheBloke/Mistral-7B-Instruct-v0.1-GGUF/tree/main Any file works but I used the 3.08 GB one <br>
Download the file and edit it to your liking <br>
Run in cmd by doing cd downloads (keep the file in downloads) and do py -3 bot.py <br>
Now try <br>
